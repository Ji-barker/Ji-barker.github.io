How to convert binary to decimal

All binary numbers have positionings but with binary numbers the positionings starts from right to left instead of left to right starting at zero with decimal.

To convert a positive binary number to a decimal 2 is powered by the positioning then times by that binary value. We then move to the next binary position and its value and repeat the first process adding together each time until we have reached the the total length of the binary number

For example converting 1010 to decimal

position 3 2 1 0
binary number 1 0 1 0

Step one
2 to the power of the position of last binary number times by the binary number position value. The position being 3 and the binary value being 1. So it would look like (2^3 * 1) this is the binary number 1000 into decimal.

Step two
Repeat step one but with the next binary number position and its value, which the position being 2 and the binary number position value being 0. So it should look like (2^2 * 0) this is the binary number of 000 into decimal.

Step three
The position being 1 and its value being 1 should look like this, (2^1 * 1) this is the binary number of 10 into decimal

Step four
The position being 0 and its value being 0 should look like this, (2^1 * 0) this is the binary number of 0 into decimal

Step five
We then add all those together, (2^3 * 1) + (2^2 * 0) + (2^1 * 1) + (2^0* 0) which equals 10. You can check this with my converter up above. (2^3 * 1) + (2^2 * 0) + (2^1 * 1) + (2^0* 0) = 10
